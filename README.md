## 🛠️ Stack technique

| Outil | Version | Rôle |
|---|---|---|
| Apache Airflow | 3.2.1 | Orchestration ETL |
| PostgreSQL | 16 | Base de données |
| Docker Compose | v5.1.3 | Conteneurisation |
| Streamlit | latest | Dashboard interactif |
| Power BI | Desktop | Visualisation avancée |
| CoinGecko API | v3 | Source données crypto |
| Python | 3.13 | Traitement données |

## 📈 Fonctionnalités

- ✅ Collecte automatique toutes les **5 minutes**
- ✅ **10 cryptomonnaies** suivies en temps réel
- ✅ Stockage dans **PostgreSQL + CSV + Excel**
- ✅ **Dashboard Streamlit** interactif
- ✅ **Dashboard Power BI** connecté à PostgreSQL
- ✅ **Alertes email Gmail** automatiques
- ✅ Gestion d'erreurs et logs avancés
- ✅ Pipeline ETL complet : Extract → Transform → Load
